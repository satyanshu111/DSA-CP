# Boundary Traversal

**Category:** Binary Trees  
**Platform:** GFG
**Link:** https://www.geeksforgeeks.org/problems/boundary-traversal-of-binary-tree/1

## Intuition
We can solve this question in 4 parts
1. push root
2. left(if exists) if not then right then again hunt for left
3. right(if exists) if not go left then again hunt for right

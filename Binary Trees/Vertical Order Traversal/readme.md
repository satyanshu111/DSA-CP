# Vertical Order Traversal

**Category:** Binary Trees  
**Platform:** LeetCode  
**Link:** https://leetcode.com/problems/vertical-order-traversal-of-a-binary-tree/

## Intuition
1. Think like a  number line, on going left cur-1, on right cur+1
2. store in map two things for each cur level and node val

## Complexity
- **Time:** O(Nlog(N))  
- **Space:** O(N)
